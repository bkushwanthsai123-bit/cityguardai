# Model Evaluation Metrics

_Placeholder. This file is generated by `ml/evaluate.py`._

Run, from the project root, after placing trained weights at `ml/weights/best.pt`:

```bash
python ml/evaluate.py --weights ml/weights/best.pt --data ml/dataset.yaml
```

It overwrites this file with the overall metrics (mAP@0.5, mAP@0.5:0.95,
precision, recall, F1), per-class AP, mean CPU inference latency (ms/image), and
embeds the confusion matrix and PR/F1 curves copied into `docs/`.

**Target:** mAP@0.5 >= 0.85 on the test split.
